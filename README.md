''Nova Basic
'' Simple Basic programming language interpreter for Linux/UNIX Terminal
''Commands
PRINT: Print text on the screen
LIST: List the program or a specific line
LET: store variable to the memory
REM:  save a comment on the instruction
RUN:	execute the instructions
LOAD:	load the program to the memory
SAVE:	save the program to a file .bas
GOTO: 	move to a specific line
